# Archived Logo (v1 — "mishti droplet" identity)

Snapshot of the original MishtiLabs brand identity, preserved before the
2026 rebrand to the **pixel-forge "M"** identity ("Engineering the Future").

| File                                | Purpose                                             |
| ----------------------------------- | --------------------------------------------------- |
| `public/icon.svg`                   | Static favicon used in `<link rel="icon">`          |
| `src/app/icon.tsx`                  | Dynamic 64×64 PNG generated by `next/og`            |
| `src/app/apple-icon.tsx`            | Dynamic 180×180 Apple touch icon                    |
| `src/app/opengraph-image.tsx`       | Social share image (1200×630)                       |
| `src/components/logo.tsx`           | Animated `<Logo>` React component (orbital + drop)  |

## Old palette

```
--accent      #c2410c   /* deep saffron */
--accent-2    #d97706   /* warm gold    */
--accent-3    #be185d   /* mishti rose  */
--highlight   #0e7490   /* cyan ink     */
```

## Old marks

- An "M" zigzag traced as a continuous gradient stroke
- A warm drop ("mishti") at the heart
- Slow + counter-rotating orbital rings
- A pulsing core circle
